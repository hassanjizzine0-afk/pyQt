
# 🔌 Modbus RTU Парсер — GUI приложение на PyQt5

## 📌 Описание проекта

Данное приложение предназначено для **анализа и расшифровки Modbus RTU пакетов** в реальном времени. Оно позволяет инженерам, разработчикам и специалистам по автоматизации быстро разбирать hex-строки, полученные от промышленного оборудования (преобразователи частоты, датчики, PLC), и получать человекочитаемый отчёт.

Приложение разработано на **Python** с использованием **PyQt5** и не требует подключения к реальному оборудованию — достаточно вставить hex-пакет.

---

## 🎯 Функциональность

| Функция | Описание |
|---------|----------|
| ✅ **Разбор пакета** | Парсинг Modbus RTU пакета на составные части |

| ✅ **Примеры пакетов** | Встроенные тестовые пакеты для проверки |
| ✅ **Копирование отчёта** | Копирование отчёта в буфер обмена |
| ✅ **Обработка ошибок** | Проверка корректности введённых данных |

### 3. Формат Modbus RTU пакета

```
┌──────────┬──────────┬──────────────┬──────────┐
│  Адрес   │  Функция │    Данные    │   CRC    │
│ (1 байт) │ (1 байт) │  (N байт)    │ (2 байта)│
├──────────┼──────────┼──────────────┼──────────┤
│   0x01   │  0x03    │  00 7E 00 02 │  25 D0   │
└──────────┴──────────┴──────────────┴──────────┘
```

### 4. Поддерживаемые функции Modbus

| Код | Название | Описание |
|-----|----------|----------|
| 0x01 | Read Coils | Чтение битовых выходов |
| 0x02 | Read Discrete Inputs | Чтение дискретных входов |
| 0x03 | Read Holding Registers | Чтение регистров хранения |
| 0x04 | Read Input Registers | Чтение входных регистров |
| 0x05 | Write Single Coil | Запись одного бита |
| 0x06 | Write Single Register | Запись одного регистра |
| 0x0F | Write Multiple Coils | Запись нескольких битов |
| 0x10 | Write Multiple Registers | Запись нескольких регистров |

### 5. Формула CRC16 (Modbus)

```
Алгоритм CRC16 Modbus:

1. Инициализация: CRC = 0xFFFF
2. Для каждого байта данных:
   a. CRC = CRC XOR байт
   b. Повторить 8 раз:
      - Если CRC & 0x0001: CRC = (CRC >> 1) XOR 0xA001
      - Иначе: CRC = CRC >> 1
3. Результат: CRC (младший байт передаётся первым)
```

## 🧪 Тестовые пакеты

| Пакет | Описание |
|-------|----------|
| `01 03 00 7E 00 02 25 D0` | Чтение 2 регистров с адреса 126 |
| `01 03 04 00 19 00 05 7A 9E` | Ответ: значения 0x0019 и 0x0005 |
| `01 90 02 CD C1` | Ошибка: Illegal Data Address |
| `01 06 00 01 00 0A 59 CD` | Запись значения 10 в регистр 1 |

## 🔧 Файлы проекта

### main.py (точка входа)
```python
import sys
from PyQt5.QtWidgets import QApplication
from gui import ModbusParserApp

if __name__ == "__main__":
    app = QApplication(sys.argv)
    window = ModbusParserApp()
    window.show()
    sys.exit(app.exec_())
```

### modbus_parser.py (модуль парсинга)
- `crc16_modbus(data)` — расчёт CRC
- `parse_modbus_packet(hex_str)` — разбор пакета
- `format_report(parsed)` — формирование отчёта

### gui.py (интерфейс)
- Класс `ModbusParserApp` — главное окно
- Методы:
  - `parse_packet()` — обработка ввода
  - `display_results()` — вывод результатов
  - `clear_all()` — очистка
  - `copy_report()` — копирование
  - `load_example()` — загрузка примера
